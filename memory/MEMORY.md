
- mybot 项目整体分为核心框架层、应用层、测试与评估层和数据与运行层  <- 1d
- core/ 包含调度器(dispatcher.py)、Agent基类(agent_base.py)、消息总线、中间件、运行器(runner.py)、编排器(orchestrator.py)  <- 1d
- context/ 包含会话上下文管理(ContextManager、Session、TokenBudget、CompactionService)  <- 1d
- agents/ 自动发现所有 BaseAgent 子类，并按 paradigm 注册(react/plan_solve 等)  <- 1d
- tools/ 包含文件操作、bash、网页抓取、搜索、子Agent委托、记忆工具、MCP集成、沙箱等工具系统  <- 1d
- providers/ 提供 LLM 提供商抽象，含 OpenAICompatibleProvider 兼容 OpenAI API 格式  <- 1d
- services/ 含 CronScheduler 定时任务调度器  <- 1d
- memory/ 含 MemoryStore(文件I/O)、Dream(LLM驱动记忆整理)、Consolidator(摘要压缩)  <- 1d
- observability/ 提供结构化日志(Loguru)、追踪(Tracer)、指标(Metrics)、OpenTelemetry桥接、终端显示  <- 1d
- tui/ 使用 Textual 框架构建终端聊天界面 ChatApp  <- 1d
- server_web/ 提供 HTTP/WebSocket 接口的 Web 服务  <- 1d
- config/ 通过 Config 类加载环境变量/配置文件  <- 1d
- skills/ 包含 20+ 子技能(文档生成、设计、MCP构建、PDF/PPTX/XLSX 等)  <- 1d
- evals/ 采用双层评估设计：自定义任务评估 + 社区基准 BFCL/GAIA，含评分器、运行器、报告器  <- 1d
- sessions/ 按会话ID分步存储 JSON 格式的会话数据  <- 1d
- 关键文件包括 pyproject.toml、mcp_servers.json、CLAUDE.md、USER.md  <- 1d
